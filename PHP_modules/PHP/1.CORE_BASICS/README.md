# PHP Core Basics

This section covers the fundamental building blocks of PHP: Variables, Data Types, Operators, and basic String/Date manipulation.

## 1. Variables and Data Types
In PHP, variables start with a `$` sign. They are loosely typed, meaning you don't need to declare the data type explicitly.
- **String**: Sequence of characters (e.g., "iPhone 15").
- **Integer**: Whole numbers (e.g., 999).
- **Float**: Decimal numbers (e.g., 89.99).
- **Boolean**: True or False values.
- **NULL**: Represents a variable with no value.

## 2. Operators
Used to perform operations on variables:
- **Arithmetic**: `+`, `-`, `*`, `/`, `%`
- **Assignment**: `=`, `+=`, `-=`
- **Comparison**: `==`, `===`, `!=`, `>`, `<`
- **Logical**: `&&`, `||`, `!`

## 3. String Manipulation
PHP provides many functions to handle text:
- **Concatenation**: Joining strings using the `.` operator.
- **`strlen()`**: Length of a string.
- **`strtoupper()` / `strtolower()`**: Changing case.

## 4. Working with Dates
The `date()` function is used to format a local date and time.
- Example: `date('Y-m-d')` returns the current date in Year-Month-Day format.
