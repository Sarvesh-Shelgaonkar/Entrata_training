# PHP Data Structures: Arrays

Arrays are used to store multiple values in a single variable. PHP supports three main types of arrays.

## 1. Indexed Arrays
Arrays with a numeric index. The index always starts at 0.
- Example: `$colors = ["Red", "Green", "Blue"];`

## 2. Associative Arrays
Arrays that use named keys that you assign to them.
- Example: `$user = ["name" => "John", "age" => 25];`

## 3. Multidimensional Arrays
An array containing one or more arrays. This is useful for storing complex data like a list of products or employees.

## Common Array Functions
- **`count()`**: Returns the number of elements.
- **`array_push()`**: Adds an element to the end.
- **`array_keys()`**: Returns all the keys of an array.
- **`sort()` / `asort()`**: Sorts arrays.
