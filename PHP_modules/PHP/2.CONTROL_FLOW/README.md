# PHP Control Flow

Control flow allows you to direct the execution of your code based on specific conditions or by repeating tasks.

## 1. Conditional Statements
Conditionals allow your program to make decisions.
- **`if / elseif / else`**: Executes code if a condition is true.
- **`switch`**: Useful for comparing the same variable against many different values.
- **Ternary Operator**: Shorthand for simple if/else (`(cond) ? true : false`).

## 2. Boolean Logic
- `true` and `false` are the two boolean values.
- **Comparison Operators**: `==`, `!=`, `<`, `>`, `<=`, `>=`.
- **Identity Operator**: `===` checks both value AND data type.

## 3. Loops
Loops are used to execute the same block of code repeatedly.
- **`for`**: When you know exactly how many times the script should run.
- **`while`**: Repeats as long as a specified condition is true.
- **`do...while`**: Always executes the block at least once, then checks the condition.
- **`foreach`**: Specifically used for looping through arrays.
