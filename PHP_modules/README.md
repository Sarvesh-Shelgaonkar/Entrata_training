# PHP Development Modules

This directory contains a structured learning path for PHP development, covering everything from core syntax to advanced database integration and unit testing.

## 📂 Project Structure

### 1. [PHP Core & Advanced](./PHP)
Highly organized modules covering the PHP lifecycle:
- **[1. CORE_BASICS](./PHP/1.CORE_BASICS)**: Variables, Types, Operators, and Strings.
- **[2. CONTROL_FLOW](./PHP/2.CONTROL_FLOW)**: Logic, Decisions, and Loops.
- **[3. DATA_STRUCTURES](./PHP/3.DATA_STRUCTURES)**: Working with various Array types.
- **[4. FUNCTIONS_AND_MODULARITY](./PHP/4.FUNCTIONS_AND_MODULARITY)**: Reusable code and Lambdas.
- **[5. WEB_AND_FORMS](./PHP/5.WEB_AND_FORMS)**: Form handling, Sessions, and Routing.
- **[6. DATABASE_AND_STORAGE](./PHP/6.DATABASE_AND_STORAGE)**: PDO Database access and File logging.

### 2. [Advanced PHP Projects](./Advanced_PHP)
- **[URL Shortener](./Advanced_PHP/url_shortner)**: A real-world project demonstrating routing, database interaction, and environment management.

### 3. [Testing with PHPUnit](./PHPUnit)
- Automated testing examples using the PHPUnit framework to ensure code quality.

## 🚀 Key Features
- **Theoretical Documentation**: Every module includes a `README.md` explaining the concepts.
- **Practical Examples**: Hands-on `examples.php` files with real-world themes (Gadgets, Cinema, Corporate, etc.).
- **Security Focused**: Demonstrations of Prepared Statements and XSS prevention.
